# AccuKnox User Management Tests

## Project Overview
This project automates the OrangeHRM User Management module using Playwright (JavaScript).

## Tools Used
- Playwright
- JavaScript
- Node.js

## Test Scenarios Covered
- Login to application
- Navigate to Admin module
- Add new user
- Search user
- Edit user details
- Delete user
- Verify deletion

## How to Run Project
npm install
npx playwright test

## Folder Structure
- tests/
- playwright.config.js
- package.json
